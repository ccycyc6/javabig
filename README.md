# 中国象棋网络版

一个基于Java Socket编程的中国象棋网络游戏，支持双人对战和聊天功能。

## 项目结构

```
ChineseChess/
├── ChessClient.java    # 客户端程序
├── ChessServer.java    # 服务器程序
└── README.md          # 项目说明文档
```

## 功能特性

- 🎮 **完整的中国象棋规则**：包含所有棋子和移动规则
- 🌐 **网络对战**：支持TCP/IP网络连接，可实现远程对战
- 💬 **实时聊天**：游戏过程中可以与对手交流
- 🎨 **图形界面**：使用Swing开发的直观棋盘界面
- 👥 **多人支持**：支持2人对战和观战模式

## 运行环境要求

- Java 8 或更高版本
- 支持TCP/IP网络连接

## 快速开始

### 1. 启动服务器

```bash
javac ChessServer.java
java ChessServer
```

服务器将在8888端口启动，等待客户端连接。

### 2. 启动客户端

```bash
javac ChessClient.java
java ChessClient
```

启动客户端后，输入服务器地址（本地测试使用localhost）即可连接。

## 游戏说明

### 操作方式

1. **选择棋子**：点击己方棋子进行选择（选中后会有黄色边框）
2. **移动棋子**：选择棋子后点击目标位置完成移动
3. **发送消息**：在右侧聊天窗口输入消息并发送

### 游戏规则

- 红方先手，双方轮流移动
- 每次只能移动一个棋子
- 遵循标准中国象棋移动规则
- 吃掉对方"帅"或"将"即获胜

## 技术实现

### 网络架构

- **TCP Socket通信**：客户端与服务器通过Socket建立连接
- **多线程处理**：服务器为每个客户端创建独立线程
- **消息协议**：使用简单文本协议传输游戏数据和聊天信息

### 核心组件

- **ChessServer**：服务器主类，负责客户端连接管理和游戏状态同步
- **ClientHandler**：客户端处理线程，处理单个客户端的请求
- **ChessClient**：客户端GUI程序，提供棋盘显示和用户交互

### 数据传输格式

- `COLOR:红/黑/观战` - 玩家颜色分配
- `BOARD:棋盘数据,当前玩家` - 棋盘状态同步
- `MOVE:起始行,起始列,目标行,目标列` - 棋子移动指令
- `CHAT:消息内容` - 聊天消息
